# PenzaiAuth
A Simple Local Yggdrasil Authentication Service for Minecraft, implemented in Python. Work In Progress.

The Yggdrasil implementation is based on the community spec documented in [authlib-injector wiki](https://github.com/yushijinhun/authlib-injector/wiki/Yggdrasil-服务端技术规范), therefore it's compatible with [authlib-injector](https://github.com/yushijinhun/authlib-injector).

### Fun fact
Penzai(盆栽) is a traditional art originated from China that grows miniature plants in containers. In this case the Yggdrasil authentication service (the name comes from the sacred tree in Norse cosmology) is implemented in miniature, hence the name.
